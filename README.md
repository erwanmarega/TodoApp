# Gestionnaire de Tâches Symfony

Il s'agit d'un projet Symfony conçu pour gérer les tâches avec la méthode CRUD, permettant aux utilisateurs de créer, mettre à jour et supprimer des tâches avec leurs descriptions et leurs statuts. Il propose également une interface front-end simple pour créer et visualiser les tâches.

## Fonctionnalités

- **Création de tâche** : Un formulaire pour créer une nouvelle tâche avec un titre, une description et un statut.
- **Liste des tâches** : Visualiser toutes les tâches avec leurs détails et l'option de les modifier ou de les supprimer.
- **Modification de tâche** : Modifier les tâches existantes, y compris le titre, la description et le statut.
- **Suppression de tâche** : Supprimer les tâches lorsque celles-ci ne sont plus nécessaires.
